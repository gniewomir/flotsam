Flotsam

Why I built this extension

Tabs accumulate. What starts as a quick lookup turns into dozens of stale pages silently consuming memory and attention. Flotsam flips the default: tabs are temporary unless you decide otherwise. The ones you care about get anchored; the rest drift away on their own.

How it works

After you install, each tab on a domain Flotsam manages (ordinary http and https addresses) gets a countdown. The default is 10 minutes; change it in Flotsam Settings. When the countdown finishes, that tab closes automatically.

Click the toolbar icon to anchor the current tab so it stays open until you unanchor it. Click again to unanchor; the countdown applies again from there.

Only tabs in the background run their countdown. The tab you are looking at does not count down. When you switch to another tab, the one you left gets a new full countdown.

You can exclude entire domains so their tabs are never auto-closed. Add or remove exclusions in Flotsam Settings, or right-click the extension icon and use the menu for the current tab’s domain. An exclusion for example.com also covers subdomains such as www.example.com.

Flotsam does not auto-close:

  • anchored tabs;
  • pinned tabs;
  • grouped tabs (tabs in any tab group);
  • tabs on excluded domains;
  • tabs the browser treats as audible (playing sound). Muted tabs and silent playback are not treated as audible.
  • tabs outside normal web domains (anything that isn’t a standard http or https tab) — for example built-in browser pages, settings, or local files opened in the browser.

Flotsam only applies to http and https tabs.

If Flotsam closes the last tab in a window, that window closes too. That is intentional.
